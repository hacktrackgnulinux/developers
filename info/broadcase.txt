Merupakan salah satu distribusi linux yang dikhususkan untuk kebutuhan penetrasi jaringan.

Operating Sistem :
* Based : Debian Based (Next Core Independent)
* Rilis : Rolling
* Arch : i386 & amd64
* Codenama : revolution
* Available : Only Live DVD or USB (No Installer to HDD)
* Status : Tester Preview

Keunikan <<Hack|Track GNU/Linux
* Buatan Indonesia untuk Indonesia
* Gratis & Open Source
* Debian Based (next core dikembangkan mandiri (independent))
* Menu Unik mix & adaptasi menu dari (backtrack,dracos,kali)

HackTrack
* bukan distribusi linux hasil remastering, 
* tapi dibuat dengan debootstrap dari minbase debian jessie yang
* selanjutnya inti sistem operasi dikembangkan sendiri hingga akhirnya mandiri. (Independent)

Project
* HackTrack merupakan project rahasia yang dikerjakan pada pertengahan tahun 2016

Kenapa dev HackTrack mengatakan project rahasia? 
* dikarenakan dev mengerjakan 2 OS yaitu Xenta OS dan Juga Hack Track.
* Setelah berselang beberapa lama dan Xenta OS berhasil di release 
* akhirnya Hack Track pun diputuskan untuk di publikasikan

Nama
* HackTrack sendiri terinspirasi dari BackTrack linux dimana dev mulai mengenal linux dari BackTrack.
* Awalnya mulai memakai nama Linux Dark, Dark Hack, 
* Hingga Akhirnya menjadi Hack Track gnu/linux seperti sekarang ini.

HackTrack sendiri masih menggunakan sistem live CD, dimana user hanya bisa menggunakan HackTrack pada FD saja, 
mungkin jika tiba pada waktunya, akan disertakan agar bisa di install pada HD.

Tujuan 
* Dikembangkannya HackTrack sendiri didasarkan ingin menciptakan suatu distribusi linux yang hanya fokus pada live sistem 
* Dikhususkan untuk kebutuhan penetrasi jaringan sesuai wujud realisasi cita cita dev "my dream is good hacker"
